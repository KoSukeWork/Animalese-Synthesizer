# Animalese Synthesizer 《动物森友会》动物语合成器

## 算法思想

1. 利用[汉字转拼音(pypinyin)](https://github.com/mozillazg/python-pinyin)将中文转拼音，拆分出音素（声韵母）。
2. 用 sox 合成音频。

## 动物语音效要点

速度 x 1.835
音高提高一个八度以上
随机化音高 抑扬顿挫
